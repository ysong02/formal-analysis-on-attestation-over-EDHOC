# formal-analysis-on-attestation-over-EDHOC

This repository contains the formal analysis of the ongoing IETF standardization effort on remote attestation over EDHOC, as specified in the draft available [here](https://datatracker.ietf.org/doc/draft-ietf-lake-ra/01/).

In parallel, we are preparing a conference paper based on this work, the target venue is to be determined.
